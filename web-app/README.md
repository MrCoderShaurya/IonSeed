# Model Showcase Web App

A React + Vite interface for showcasing your model with live input and output.

## Run locally

1. Open a terminal in `web-app`
2. Install dependencies:
   - `npm install`
3. Start development server:
   - `npm run dev`
4. Open the URL shown in the terminal, usually `http://localhost:5173`

## What it includes

- Interactive input form
- Dummy model run logic for live demo
- Output panel for results
- Clean responsive UI
